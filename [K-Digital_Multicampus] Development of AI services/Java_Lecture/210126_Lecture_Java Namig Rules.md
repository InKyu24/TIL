# Java Namig Rules



## class { }

+ 자바 프로그램 단위

- 띄어쓰기 금지
- 첫 글자 숫자 금지
- 대문자로 시작 관례
    (ex. HelloWorld, MaxSpeed, FirstName, CarBodyColor)



> Class ⊃ data, method
>
>  Method ⊃ 행위

 

## method ( ) { }

- 소괄호를 포함(parameter, argument)
- main method는 시작 위치
- 첫번째 글자는 소문자로 시작하고, 다른 단어가 붙을 때 대문자로 하는 것이 관례
   (ex. maxSpeed, firstName, carBodyColor)

 

## Variable

- 첫번째 글자는 **문자**, **$**, **_**만 가능, 숫자는 불가능
- 대소문자 구분
- 길이 제한 없음
- 자바 예약어(Keyword) 사용 불가
- 첫번째 글자는 소문자로 시작하고, 다른 단어가 붙을 때 대문자로 하는 것이 관례
    (ex. maxSpeed, firstName, carBodyColor)
